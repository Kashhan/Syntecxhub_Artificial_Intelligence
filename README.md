# Syntecxhub_Artificial_Intelligence

## Maze Solver using A Star Search

This project implements the A Star search algorithm to find the shortest path
in a maze from a start point to a goal point.

Features
Uses grid based maze representation
Implements A Star with Manhattan heuristic
Handles unreachable goal cases
Visualizes the final path using matplotlib

How to Run
1 Install dependencies
pip install -r requirements.txt

2 Run the program
python main.py

Output
The shortest path is printed in the console
A graphical maze visualization is displayed
